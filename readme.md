## 上升对比互联网vs制造业

```
┌──────────────────────────────────────────────────────────────┐
│                IT系统架构对比（互联网 vs 制造业）             │
├───────────────┬──────────────────────────┬────────────────────┤
│ 维度           │ 互联网行业（ToC / ToB SaaS） │ 制造业（工业 / 工厂） │
├───────────────┼──────────────────────────┼────────────────────┤
│ 核心目标       │ 用户增长 / 流量 / 转化       │ 生产效率 / 质量 / 成本 │
│ 核心对象       │ 用户（User）                │ 产品 + 设备 + 人       │
│ 主流程         │ 用户行为流程                │ 生产制造流程            │
├───────────────┼──────────────────────────┼────────────────────┤
│ 系统核心       │ App + 平台系统             │ MES + ERP              │
│ 架构风格       │ 微服务 / 云原生            │ 分层 + OT/IT融合        │
│ 实时性要求     │ 毫秒级（用户体验）          │ 秒级~分钟（生产节拍）   │
├───────────────┼──────────────────────────┼────────────────────┤
│ 数据来源       │ 用户行为 / 日志            │ 设备 / PLC / 测试数据   │

├───────────────┼──────────────────────────┼────────────────────┤
│ 系统层级       │ 前端 → 后端 → 数据层        │ ERP → MES → SCADA → PLC │

├───────────────┼──────────────────────────┼────────────────────┤
│ 核心挑战       │ 高并发 / 可扩展 / SLA       │ 设备集成 / 稳定性 / 落地 │
│ 架构难点       │ 分布式一致性               │ OT/IT融合 + 实时控制     │
├───────────────┼──────────────────────────┼────────────────────┤
│ AI应用         │ 推荐 / 搜索 / 广告优化       │ 质量分析 / 预测 / 调参  │
│ 成功标准       │ DAU / 留存 / GMV           │ 良率 / OEE / 产能       │
```
